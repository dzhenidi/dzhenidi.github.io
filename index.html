Easy Data

GitHub

EasyData is a lightweight framework that facilitates SQL record manipulation through object relational mapping. It is inspired by Rails' ActiveRecord, and, similarly to ActiveRecord::Base, converts database tables into instances of the SQLObject class and encapsulates SQL queries in simple ruby operations. The library allows for single and multi-step associations between models in order to streamline common queries involving multiple tables. I followed Rails' naming conventions to provide default table and class values, equipping the associations methods with default arguments. I also used Ruby metaprogramming techniques to define getter and setter methods on the associations.

Tetronimos

live GitHub

Tetronimos is a tile-matching JS browser game that follows Object-Oriented Design to allow for easy add-on of features. It uses ES6 language specification and babel to transpile code into ES5. The animations are created with the Canvas API.
